# Эпик 09 — Панель администратора

## Контекст

Владелец продукта должен понимать, как используется платформа: сколько матчей проходит, кто играет, какова активность. Панель предоставляет только инструменты наблюдения — без вмешательства в игровой процесс.

---

## План

### Доступ
- Администратор — зарегистрированный пользователь с ролью `admin`
- При входе под admin-аккаунтом открывается панель администратора
- Панель недоступна обычным пользователям

### Активность в реальном времени
- Количество матчей, идущих прямо сейчас
- Количество игроков онлайн в данный момент
- Администратор может подключиться к наблюдению любого активного матча прямо из панели — кнопка «Наблюдать» рядом с каждым активным матчем
- Администратор видит матч в том же интерфейсе, что и обычный наблюдатель, но не входит в общий лимит двух наблюдателей на комнату
- Участники матча не уведомляются о подключении администратора

### Статистика
- Количество сыгранных матчей за день, неделю и всё время
- Разбивка по типу матча: онлайн против игрока / против бота

### Журнал матчей
- Список завершённых матчей: участники, счёт по сетам, дата и время, продолжительность
- Фильтрация журнала по дате

### Игроки
- Список зарегистрированных игроков
- Статистика каждого игрока: количество матчей, побед, поражений, процент побед
