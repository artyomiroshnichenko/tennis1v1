# Эпик 06 — Наблюдатель

## Контекст

Друзья могут хотеть наблюдать за матчем, не участвуя в нём. Наблюдатель подключается к комнате по той же ссылке, смотрит игру в реальном времени и участвует в общем чате.

---

## План

### Подключение
- Наблюдатель переходит по ссылке комнаты после старта матча — система автоматически назначает роль наблюдателя
- Если матч ещё не начался — наблюдатель видит сообщение «Матч ещё не начался» и ожидает
- Максимум 2 наблюдателя в одной комнате одновременно
- При попытке третьего наблюдателя подключиться — вход закрыт с соответствующим сообщением
- При подключении в середине матча — наблюдатель сразу видит текущее состояние корта и счёт

### Возможности наблюдателя
- Видит игру в реальном времени
- Пишет в общий чат комнаты
- Остаётся подключённым между матчами в той же комнате
- Переходит на экран результата вместе с игроками (нейтральная версия)

### Индикатор наблюдателей
- На экране игроков отображается иконка глаза с количеством наблюдателей
- При подключении нового наблюдателя счётчик увеличивается
- При отключении наблюдателя счётчик уменьшается; если наблюдателей нет — иконка исчезает

### Поведение при паузе
- Если один из игроков отключился — наблюдатель видит паузу с обратным отсчётом так же как соперник

### Ограничения наблюдателя
- Не может участвовать в матче
- Не может влиять на ход игры
- Отключение наблюдателя никак не влияет на матч

- [ ] Если матч не начался — наблюдатель видит сообщение об ожидании

- [ ] При превышении лимита — наблюдатель получает сообщение об отказе
- [ ] Наблюдатель видит паузу с отсчётом при отключении игрока

- [ ] При отсутствии наблюдателей иконка глаза исчезает
