# Эпик 04 — Экран результата и история

## Контекст

После завершения матча игроки и наблюдатели видят итог. Зарегистрированные пользователи имеют доступ к истории своих матчей в профиле. Гости видят только результат текущего матча.

---

## План

### Экран результата
- Отображается полный счёт по сетам и победитель матча
- Экран персональный: победителю — оптимистичное оформление (зелёная цветовая схема), проигравшему — сдержанное (красная)
- Наблюдатели также переходят на экран результата по завершении матча — видят нейтральную версию без цветовой окраски
- Матч завершённый по отключению соперника — фиксируется как техническое поражение для отключившегося

### Действия после матча
- Доступны два действия: «Сыграть ещё раз» и «В главное меню»
- «Сыграть ещё раз» требует согласия обоих игроков — каждый должен нажать кнопку
- Если один игрок вышел в меню — комната остаётся открытой, он может вернуться по той же ссылке
- При возвращении оба снова могут нажать «Сыграть ещё раз»
- При подтверждении обоих — комната остаётся той же, состояние матча сбрасывается

### История матчей
- Результат каждого матча автоматически сохраняется в историю зарегистрированного пользователя
- Сохраняются последние 100 матчей
- История доступна на отдельной странице в профиле
- Каждая запись содержит: соперник, счёт по сетам, дата, исход (победа / поражение / техническое поражение)
- Гость не имеет доступа к истории
